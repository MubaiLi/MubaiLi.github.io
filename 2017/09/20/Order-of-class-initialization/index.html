类的初始化顺序

发表于 2017-09-20 | 分类于 Java基础知识

在创建类的过程中，各成员有一个执行顺序：

  1. 执行超类（父类）中的静态成员和代码块
  2. 执行子类中的静态成员和代码块
  3. 执行超类的实例成员和代码块
  4. 执行超类的构造函数
  5. 执行子类的实例成员和代码
  6. 执行子类的构造函数
